Flexbox and Grid: Modern Layouts

1. CSS Flexbox

The **Flexible Box Layout Module**, or Flexbox, is a one-dimensional layout system used to arrange items in a single row or column. It makes it easy to align items, distribute space, and reorder elements.

<div class="flex flex-col md:flex-row justify-center items-center">
    <div class="p-4 bg-gray-200">Item 1</div>
    <div class="p-4 bg-gray-300">Item 2</div>
</div>

2. CSS Grid Layout

CSS Grid is a powerful **two-dimensional** layout system that handles both rows and columns at the same time. It's ideal for building complex, responsive page layouts.

<div class="grid grid-cols-2 md:grid-cols-3 gap-4">
    <div class="p-4 bg-blue-200">Cell 1</div>
    <div class="p-4 bg-blue-300">Cell 2</div>
    <div class="p-4 bg-blue-400">Cell 3</div>
</div>

3. Choosing the Right Tool

The key difference is dimensionality. Use **Flexbox** for aligning items in a single row or column. Use **Grid** for creating complex, two-dimensional layouts.

  • **Flexbox** is best for components like navbars, forms, and cards.
  • **Grid** is best for overall page structure, like a header, sidebar, and main content area.
