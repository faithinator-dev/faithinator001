JavaScript: The Language of the Web

1. What is JavaScript?

JavaScript is the programming language that makes websites interactive. It allows you to create dynamic content, respond to user actions (like clicks), and even build entire web applications. While HTML provides the structure and CSS provides the style, JavaScript provides the **behavior**.

// This is a comment in JavaScript
console.log("Hello, World!");

2. Variables and Data Types

A **variable** is a container for storing a value. JavaScript has several data types, including strings (text), numbers, booleans (true/false), and objects.

var name = "Alice"; // A string
let age = 30; // A number
const isStudent = false; // A boolean
let person = {
  firstName: "John",
  lastName: "Doe"
}; // An object

3. Creating and Using Functions

A **function** is a block of code designed to perform a particular task. It allows you to reuse code without having to write it multiple times.

function greet(name) {
  return "Hello, " + name + "!";
}

let message = greet("Bob");
console.log(message); // Output: Hello, Bob!
