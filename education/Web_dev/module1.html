HTML5: The Web's Foundation

1. What is HTML5?

HTML5 is the latest version of **HyperText Markup Language**, the code used to structure content on the web. It provides a common language for all browsers, allowing them to render text, images, videos, and other content correctly. Think of it as the **skeleton** of a website—it gives the page its form, but no style.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First HTML5 Page</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

2. Essential Elements (Tags)

HTML uses **tags** to create elements. A tag is just a keyword surrounded by angle brackets, like `

`. Tags usually come in pairs: an opening tag and a closing tag.

  • <h1> to <h6> for headings.
  • <p> for a paragraph.
  • <a> for a link (anchor tag).
  • <img> for an image (self-closing tag).
  • <ul> and <ol> for lists.
<h1>Welcome to My Site</h1>
<p>This is a paragraph with a <b>bold</b> word.</p>
<a href="https://example.com">Learn more</a>
<img src="placeholder.png" alt="A descriptive image">
<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

3. The Power of Semantic Tags

A major advancement in HTML5 was the introduction of **semantic tags**. These tags don't just structure the page, they also describe the type of content they contain. This is crucial for search engines and accessibility tools.

  • <header>: The top section of a page or article.
  • <nav>: A section containing navigation links.
  • <main>: The main content of the document.
  • <article>: Self-contained, independent content (like a blog post).
  • <footer>: The bottom section of a page.
<body>
    <header>
        <h1>My Blog</h1>
        <nav>
            <a href="#">Home</a>
            <a href="#">About</a>
        </nav>
    </header>
    <main>
        <article>
            <h2>My First Blog Post</h2>
            <p>Content goes here...</p>
        </article>
    </main>
    <footer>
        <p>© 2025 All rights reserved.</p>
    </footer>
</body>

4. Forms and User Interaction

HTML forms are how you get information from a user. This includes text fields, checkboxes, buttons, and more. The `

` tag wraps all the input elements.

<form action="/submit-form" method="post">
    <label for="name">Your Name:</label><br>
    <input type="text" id="name" name="name"><br><br>
    
    <label for="email">Your Email:</label><br>
    <input type="email" id="email" name="email"><br><br>
    
    <button type="submit">Submit</button>
</form>
