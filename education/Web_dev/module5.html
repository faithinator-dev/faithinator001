ES6 and Beyond: Modern JavaScript

1. Variable Declarations: `let` and `const`

Before ES6, `var` was the only way to declare variables. Now, `let` and `const` are the preferred choices. `let` is used for variables that can be reassigned, while `const` is for variables that cannot be changed once they are declared. This helps prevent bugs and makes your code more predictable.

let count = 0;
count = 1; // OK

const PI = 3.14;
// PI = 3.14159; // This would cause an error!

2. The Arrow Function Syntax

Arrow functions provide a shorter, cleaner syntax for writing functions. They are especially useful for simple, one-line functions.

// Traditional function
const add = function(a, b) {
  return a + b;
};

// Arrow function
const addArrow = (a, b) => a + b;

3. Template Literals

Template literals (using backticks `` ` ``) allow you to embed expressions directly into strings. This makes string concatenation much easier to read and write.

const name = "Jane";
const greeting = `Hello, ${name}! Welcome.`;
console.log(greeting); // Output: Hello, Jane! Welcome.
